Última actualización: junio de 2026

1. Descripción del servicio

AutomatizaPyme es una aplicación de escritorio para Windows que integra un ERP (facturación, contabilidad, RRHH, inventario, etc.) con agentes de inteligencia artificial. El software se instala localmente en el equipo del usuario; los datos del ERP se almacenan en una base de datos PostgreSQL local.

  • Usar el software para fines ilegales o fraudulentos.

4. Modelo BYOK (trae tu propia clave de IA)

Las funciones de inteligencia artificial requieren una clave API de un proveedor externo (Anthropic, OpenAI, Google, etc.) que el usuario configura y paga directamente. AutomatizaPyme no intermedia ni factura el consumo de IA. El usuario es responsable de cumplir los términos de uso del proveedor LLM que elija.
