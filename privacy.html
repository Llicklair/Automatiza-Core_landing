Política de Privacidad

Última actualización: junio de 2026

En AutomatizaCore nos tomamos muy en serio la privacidad de nuestros usuarios. Esta política describe qué datos recopilamos, cómo los usamos y cómo los protegemos.

1. Responsable del tratamiento

Marcos Recio Sánchez
Correo de contacto: marcosreciosanchez@gmail.com

2. Datos que recopilamos

AutomatizaCore es una aplicación de escritorio que procesa datos localmente en el ordenador del usuario. Los datos que podemos recopilar son:

  • Datos de cuenta Google: cuando el usuario conecta su cuenta de Gmail u otros servicios de Google, accedemos únicamente a los permisos que el usuario autoriza explícitamente (lectura y envío de correos).
  • Datos de contacto: nombre y correo electrónico cuando solicitas una demo o nos contactas por email.
  • Datos empresariales: facturas, contactos, empleados y otros datos del ERP se almacenan localmente en la base de datos del propio usuario, no en nuestros servidores.

3. Uso de los datos

Los datos recopilados se usan exclusivamente para:

  • Prestar el servicio de automatización solicitado por el usuario (lectura/envío de correos, sincronización de calendario, etc.).
  • Responder a solicitudes de demo o soporte técnico.
  • Mejorar el producto mediante métricas de uso agregadas y anónimas.

No vendemos, cedemos ni compartimos datos personales con terceros con fines comerciales.

4. Integración con Google (Gmail)

AutomatizaCore utiliza la API de Google para ofrecer funcionalidades de correo electrónico. El uso que hacemos de la información recibida de las APIs de Google cumple con la Política de Datos de Usuario de los Servicios de API de Google, incluidos los requisitos de uso limitado.

  • Solo accedemos a los datos de Google que el usuario autoriza explícitamente.
  • No usamos los datos de Google para mostrar publicidad.
  • No transferimos datos de Google a terceros salvo cuando sea necesario para prestar el servicio solicitado.
  • El usuario puede revocar el acceso en cualquier momento desde su cuenta de Google.

5. Almacenamiento y seguridad

Los datos del ERP se almacenan en la base de datos local del usuario (PostgreSQL en el propio equipo). Las credenciales sensibles (tokens OAuth, API keys) se cifran mediante el sistema de almacenamiento seguro del sistema operativo (Windows Credential Manager / Electron safeStorage).

6. Derechos del usuario

De acuerdo con el RGPD, el usuario tiene derecho a:

  • Acceder, rectificar o suprimir sus datos personales.
  • Oponerse al tratamiento o solicitar su limitación.
  • Portabilidad de los datos.

Para ejercer estos derechos, contacta con nosotros en marcosreciosanchez@gmail.com.

7. Cambios en esta política

Podemos actualizar esta política ocasionalmente. La fecha de la última actualización siempre aparecerá al inicio de este documento. El uso continuado de la aplicación tras los cambios implica la aceptación de la nueva versión.
